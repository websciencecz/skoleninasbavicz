Docker - ovládni kontejnery jako profík

Termín: ---

Pro koho je školení určeno:

Pro ty, kteří chtějí poznat sílu Dockeru. Naučit se, jak sestavit image, jak ovládat kontejnery a spravovat je za pomoci Docker Compose. Připravíme si sandbox aplikaci, kde si ukážeme jak nastavit vývoj multi-kontejnerové aplikace s MySQL, Elasticsearch nebo Redis.

Co se na školení naučíte:


Jak Docker funguje a jak jej jednoduše a efektivně používat
  • Nainstalujeme Docker a Docker Compose (může být již nainstalovaný)
  • Seznámíme se s základními pojmy a ovládáním
  • Co je to image, kontejner, volume, expose atp.
  • Jak pracovat s kontejnery a využít je na maximum

Docker Compose a Docker Swarm
  • Ukážeme si, jak provozovat multi kontejnerovou aplikaci
  • Jak dostat data do kontejneru a zase je z něj získat
  • Jak funguje komunikace mezi kontejnery ve stejné síti a jak se k nim dostat z "venku"
  • Vyzkoušíme si provoz aplikace za pomoci Docker Swarm (provoz aplikace v clusteru)

Nginx Proxy
  • Přístup k aplikaci na localu skrze doménu
  • HTTPS na localhostu
  • Layer 7 loadbalancer

Co budete potřebovat:

  • Základní znalost Linuxu
  • Notebook
  • Docker a Docker Compose https://docs.docker.com/install/
