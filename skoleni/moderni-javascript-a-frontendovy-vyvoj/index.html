Moderní JavaScript a front-endový vývoj

Termín: ---

Pro koho je školení určeno:

Určeno pro všechny, pro něž je vstupní bariéra do ekosystému moderního Javascriptu příliš vysoká a neprostupná: pro frontenďáky, kteří se zasekli u slepování jQuery pluginů a chtěli by se posunout dál anebo se už posouvají, ale tápou při tom; pro backenďáky, kteří si chtějí rozšířit obzory.

Co se na školení naučíte:


Moderní Javascript

Ukážeme si, kam se posunul a neustále posouvá Javascript jako jazyk. Novinek je spousta a od roku 2015 přibývají pravidelně každý rok. Projdeme si ty nejzásadnější od syntaktických cukrátek přes třídy a moduly až po Promises a async/await.


Moderní front-end

Proletíme novinky, které se naučily webové prohlížeče. Zahodíme dolary z jQuery a nahradíme je moderními API pro manipulaci s DOMem. A asynchronní HTTP požadavky budeme posílat pomocí jedné jediné jednoduché funkce.


Moderní build

Naučíme se pracovat se správcem závislostí NPM a sestavíme si s pomocí Webpacku celou aplikaci dohromady tak, aby byl spokojený jak vývojář, tak uživatelé v produkci. A celé to bude provozuschopné i v Internet Exploreru.


Co budete potřebovat:

Prerekvizitami budiž alespoň základní znalost Javascriptu a nainstalované Node.js (https://nodejs.org/en/download)
